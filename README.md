## About

I design and implement **AI-driven automation systems** to improve sales, operations, and customer engagement. With hands-on experience in both **B2B sales and technology**, I bridge the gap between business needs and scalable solutions.

I focus on solving real-world problems using **automation, data, and AI** — building systems that are not just technically sound, but genuinely useful to the people using them.

```
  AI Automation   →   Smarter workflows, less manual effort
  Data Analysis   →   Insights that drive better decisions  
  B2B Sales       →   I understand the customer and can sell the solution
```
